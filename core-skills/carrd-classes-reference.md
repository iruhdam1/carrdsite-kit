# Carrd Classes Reference

Quick reference for all custom classes used with carrd-skills. Add these via the element's gear icon → Advanced → Classes.

## How to Add a Class

1. Click the element in the Carrd editor
2. Click the gear icon (⚙️)
3. Go to **Advanced → Classes**
4. Click **Add class** and type the class name

---

## Navigation

| Class | Skill | Effect |
|---|---|---|
| `sticky-nav` | sticky-nav | Makes the container stick to the top on scroll |
| `nav-link` | sticky-nav | Identifies individual nav link elements |

## Animations

| Class | Skill | Effect |
|---|---|---|
| `fade-in` | scroll-animations | Fades element in when scrolled into view |
| `slide-up` | scroll-animations | Slides element up 40px into view |
| `slide-left` | scroll-animations | Slides element in from left |
| `slide-right` | scroll-animations | Slides element in from right |
| `scale-in` | scroll-animations | Scales element from 90% to 100% |

## Interactive Components

| Class | Skill | Effect |
|---|---|---|
| `faq` | faq-accordion | Marks the container holding FAQ content |
| `typing` | typing-effect | Animates text with typewriter effect |
| `popup-trigger` | popup-modal | Button/element that opens the modal |
| `draggable` | draggable-containers | Makes the container draggable by visitors |

## Theming

| Class | Skill | Effect |
|---|---|---|
| `card` | dark-mode-toggle | Container changes background in dark mode |
| `footer` | dark-mode-toggle | Excludes container from dark mode background changes |

---

## Notes

- Classes are case-sensitive — use exactly as written above
- Multiple classes can be applied to one element (space-separated in the Classes field)
- Some skills generate their own elements and don't require any classes
